SlackBot-Links

This is a simple REST API allowing consumers to search links shared on Slack.

To install this bot, follow installation instructions.

All Links

Search for a Link

REQUEST

GET

http:/yourdomain.com/v1/links{?limit}{?offset}{?search}{?url}
Parameters
limit number

The maximum number of results to return.
offset number

Return results from this offset.
search string

Search by term.
url string

Search by url.
sharedby string

Filter links shared by this user.

Headers
Content-Type application/json
200 application/json
